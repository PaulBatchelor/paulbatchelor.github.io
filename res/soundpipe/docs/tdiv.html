tdiv
Files: tdiv.h, tdiv.c

Trigger divider. This module will take in a trigger signal, and output a trigger signal every N times.
For instance, when N = 3:
in: * * * * * * * * *
out * * *

Functions

sp_tdiv_create(sp_tdiv **tdiv)
sp_tdiv_init(sp_data *sp, sp_tdiv *tdiv)
sp_tdiv_compute(sp_data *sp, sp_tdiv *tdiv, SPFLOAT *trigger, SPFLOAT *out)
sp_tdiv_destroy(sp_tdiv **tdiv)

Optional Parameters

num: Triggers every N times.
(Default value: 2)
offset: Offset amoutn.
(Default value: 0)

Inputs

trigger: Trigger. Should be a non-zero value

Outputs

out: Signal output.
