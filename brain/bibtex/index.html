Casual notes on BibTex, so I don't forget.

With any luck, this actual org file is contained in a folder called bibtex.

At the time of writing, I have created an initial janet script called bibgen.janet that generates a bibtex file from my sqlite table and writes it to standard output.

Go into the bibtex folder (assuming I don't move this) and run it like this: 

weewiki janet bibgen.janet > test.bib

This will redirect standard output to test.bib.

There should be a script in the bibtex folder called test.sh, run that script to generate a test pdf which should generate the entire bibliography. It is also worth looking at the script itself. Both tex and bibtex apparently need to be run twice in order to update the backlinks properly.
