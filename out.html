A bootstrap setup

Note: now that sync is utilized, this should not be used.
The following describes a bootleg weewiki sync setup usingonly push/pull commands and shell scripts.

mkwiki

On a new system, the wiki database must be bootstrapped.
This will take all the external org files and pushthem to the database.

push

A mass push operation will update the internal pages to bethe external pages.

pull

A mass pull operation will update all external pagesto be what is stored internally.
