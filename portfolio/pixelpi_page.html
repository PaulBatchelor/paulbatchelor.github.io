Pixel Pi

PixelPi is a mini-sound installation created by myself and Christopher Konopka. It runs entirely off a Raspberry Pi computer with a Raspberry Pi camera attachment. It is written using Python and Csound.

In PixelPi, a picture is periodically taken. The RGB and HSL data is then analyzed and then used as parameters for a generative composition synthesized in realtime.

PixelPi was first comissioned for the 2013 Csound Conference hosted Berklee College of Music, Boston. It was then later redesigned from the ground up for the 2014 Boston A3e conference.

My work in PixelPi was to design the algorithms and sounds that would work in realtime.
