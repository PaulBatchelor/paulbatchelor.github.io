Pixel Pi

Pixel Pi was a micro-installation created by myself and Christopher Konopka for the 2014 International Csound Conference, and then recreated for the A3e conference.

Pixel Pi at A3e is shown below:

PixelPi is designed to run entirely on a Raspberry Pi with a Raspberry Pi camera. It periodically takes pictures using the camera, and then sonifies it using and internal algorithm built using Pythong, PIL, and Csound.
