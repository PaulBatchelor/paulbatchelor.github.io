1-bit dither

1-bit dithering, such as Atkinson Dither, can be used as an aesthetic. The monolith janet function monolith/gfx-dither takes a region in the framebuffer and applies an Atkinson Dither filter using two arbitrary RGB colors.

The arguments are x, y, width, height, RGB color 1, and RGB color 2.

The following janet code creates a 1-bit dither region of size 16x64 at position (32,0) with the colors blood red (102, 0, 0), and NCS red (192, 2, 51). 

(monolith/gfx-dither
 32 0 16 64
 # Blood Red
 102 0 0
 # NCS red
 192 2 51)

More information on the implemtnation can be found at dither.c, with the main algorithm outlined at 4. The Dither Algorithm.
