24. thresh

thresh is a threshold generator. It will output a tick when an input signal in goes past a threshold thr. The type flag type determines the threshold behavior.

type flags:

0: from below, 1: from above, 2: from above/below

<<ugens-top>>=
(define (thresh in thr type)
  (param in)
  (param thr)
  (param type)
  (rvl "thresh"))
