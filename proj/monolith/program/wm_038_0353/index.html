84. cthrow

cthrow "throws", or mixes, a signal into a summation cable to be used later on in the patch. The mix amount amt is typically a 0-1 that shows how much of the value to mix in.

cab is a lambda which should retrieve the send cable. This is typically done with cabget.

<<ugens-top>>=
(define (cthrow in amt cab)
  (param in)
  (param amt)
  (cab)
  (rvl "cthrow"))
