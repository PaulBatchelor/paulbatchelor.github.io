4. Error

Wrapper for the s9 scheme error function. Utilized so that s9ext.h does not have to be used.

<<scheme_function_declarations>>=
s9_cell monolith_scheme_error(char *msg, s9_cell expr);
<<scheme_functions>>=
s9_cell monolith_scheme_error(char *msg, s9_cell expr)
{
    return error(msg, expr);
}
