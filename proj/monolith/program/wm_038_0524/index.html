127. wpkorg35

wpkorg35 is a korg lowpass filter design based off an implementaton by will pirkle.

It takes in an input signal in, a filter cutoff cutoff in Hz, a resonance res, which is a value between 0 and 2, and saturation amount sat.

<<ugens-top>>=
(define (wpkorg35 in cutoff res sat)
  (param in)
  (param cutoff)
  (param res)
  (param sat)
  (rvl "wpkorg35"))
