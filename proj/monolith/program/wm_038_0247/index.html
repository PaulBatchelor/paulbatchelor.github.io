58. tdiv

tdiv is a trigger divider. Given an incoming trigger signal in, it will tick every num ticks. offset is a value between 0 and num - 1 that can be used to shift when the tick happens (this can be used to create syncopation for things like backbeats).

<<ugens-top>>=
(define (tdiv in num offset)
  (param in)
  (param num)
  (param offset)
  (rvl "tdiv"))
