86. tcross

tcross is a 3-way table crossfader. pos provides the wavetable position as a normalized value. in is the lookup position, which is also expected to be a normalized value.

<<ugens-top>>=
(define (tcross in pos fta ftb ftc)
  (param in)
  (param pos)
  (fta)
  (ftb)
  (ftc)
  (rvl "tcross"))
