47. crossfade

crossfade performs a crossfade operation between signals sig1 and sig2 with a crossfade value pos. When pos is 0, it is all sig1, and when it is 1 it is all sig2. 0.5 is an equal mix.

<<ugens-top>>=
(define (crossfade sig1 sig2 pos)
  (param sig1)
  (param sig2)
  (param pos)
  (rvl "crossfade"))
