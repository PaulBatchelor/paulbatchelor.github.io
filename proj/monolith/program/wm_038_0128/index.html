28. tport

tport is a portamento filter with a triggerable reset. It is similar to port, execept that it takes in a trigger signal trig. When a trigger happens the filter resets. This is useful for instances when you want to start a note without it sliding from a previous note.

<<ugens-top>>=
(define (tport in trig htime)
  (param in)
  (param trig)
  (param htime)
  (rvl "tport"))
