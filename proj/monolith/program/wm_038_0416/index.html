100. regclr

Clears a runt register in position r. if a negative value is provided, it clears all of them. Explicitely calling regclr with bunholdall is a good idea in a live coding setup incase a patch doesn't get fully evaluated.

<<ugens-top>>=
(define (regclr r)
  (param r)
  (rvl "regclr"))
