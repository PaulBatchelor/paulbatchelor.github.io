15. Janet

Janet can be run as a standalone program with weewiki janet. It is almost identical to the vanilla janet program, except that the weewiki functions are loaded as well.

<<argparse_entries>>=
{"janet", 5, p_janet, NULL},
<<static_funcdefs>>=
static int p_janet(int argc, char *argv[]);
<<functions>>=
int janet_main(int argc, char *argv[]);
static int p_janet(int argc, char *argv[])
{
    return janet_main(argc, argv);
}
