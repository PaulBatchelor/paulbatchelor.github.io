24. Crate

The crate is a interface that connects the SQLar format to the weewiki zettelkasten.

<<argparse_entries>>=
{"crate", 5, p_crate, NULL},
<<static_funcdefs>>=
static int p_crate(int argc, char *argv[]);
<<functions>>=
int ww_crate(int argc, char *argv[]);
static int p_crate(int argc, char *argv[])
{
    return ww_crate(argc, argv);
}
