7. Fill

The function btprnt_fill will fill a region.

<<funcdefs>>=
void btprnt_fill(btprnt_region *reg, int clr);
<<funcs>>=
void btprnt_fill(btprnt_region *reg, int clr)
{
    int x, y;
    for (y = 0; y < reg->h; y++) {
        for (x = 0; x < reg->w; x++) {
            btprnt_region_draw(reg, x, y, clr);
        }
    }
}
