Orgdown

Orgdown is the name of the markup language used by org-mode in Emacs.

The weewiki wiki engine uses it's own flavor of orgdown.

The worgle org tangler for literate programming also expects a subset orgdown input, required for programs that woven to the loom. Otherwise, worgle only expects code blocks written in a noweb style.

More information on orgdown can be found at: https://gitlab.com/publicvoit/orgdown
