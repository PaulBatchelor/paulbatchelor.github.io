RG350

The RG-350 is a portable gaming device running OpenDingux Linux.

These are my notes.

You can SSH into the thing as root using 

ssh root@10.1.1.2

On my computer, the subnets were already in place for whatever reason and I could get in just fine.

Be sure to use the right USB port to plug in.

Using a USB C adapter, things like keyboard and mice work with generic HID drivers.

Other devices (monome grid, USB MIDI, etc) don't seem to work.
