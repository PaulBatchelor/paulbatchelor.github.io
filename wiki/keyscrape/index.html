keyscrape

keyscrape is a utility found in weewiki. This program will go through every weewiki page and look for words and phrases using the verbatim tag (aka = and =). Their location is the printed to a tab-separated file (known as a TSV) for further processing.

The keyscrape command is usually used to make a keywords page, like the one found on this wiki.

Keyscrape is written in a literate style. The woven part of the literate programcan be read online via the loom
