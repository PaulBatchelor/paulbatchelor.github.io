Plans for Toys

toys not tools is a mantra I recently have adapated for myself, and hope to explore in 2021. Here are some tasks I have set for myself, populated in zetdo found in this wiki.

brainstorm: strategize bringing back voc and speech synthesis as a toy.

add a toy language to sndkit

embed sndkit algorithms in a usable toy language (thinking about (LIL)) that would allow one to build up patches using (patchwerk).

bring back android interactive audio projects

this is a planning-to-action task. generally, it is about setting up a manageable toolchain for making interactive audio apps using just C, NDK, and nanovg. There are some interesting makefile android projects that don't use ndkbuild that look promising.
