List of Starting Verbs in Metafont

Knuth is (unsurprisingly) consistent in his literate programs. Leafing through Metafont: The Program (Book D in the Computers and Typsetting Series), I discovered that almost all the named codeblocks followed a similar grammatical structure that started in a verb. Examples include "Append the current expression to arg_list", "Back up an outer symbolic token so that it can be reread", and "Compute a compromise pen_edge". Many of these verbs get re-used, so I thought it would be neat to write them all down.
