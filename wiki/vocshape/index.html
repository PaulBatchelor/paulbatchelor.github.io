VocShape

Vocshape is a small physically based vocal synthesis model that runs on an android phone, that is capable of producing sounds not unlike a throat singer.

The app was written as a small demo for a rejected PhD application of mine..

The interface allows one to directly "sculpt" or shape the underlying vocal tract (44 discrete diameters), and consequently manipulating the vocal-like timbres.

When two fingers are pressed down, the program allows you to change the pitch of the instrument.

Source code can be found on sourcehut.
