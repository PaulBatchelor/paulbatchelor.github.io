Mnodes

The mnodes project is a collection of miscellaneous DSP algorithms for mnolth, using the sndkit C API for making nodes. For various reasons, these aren't included in the main sndkit distribution.

To use with mnolth, make sure the mnodes directory is copied or symlinked to mnolth/mnodes, and that the "mnodes" configuration option is set. For example: 

$ lua generate_ninja.lua mnodes
