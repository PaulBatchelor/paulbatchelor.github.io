FFMPEG

Some various ffmpeg commands that might be useful for me later.

fade in/out 

ffmpeg -y -i voctest_v2.mp4 \
    -ss 00:01 -to 00:35 \
    -vf "fade=t=in:st=1:d=3,fade=t=out:st=33:d=2" \
    out.mp4

sources https://blog.feurious.com/add-fade-in-and-fade-out-effects-with-ffmpeg
